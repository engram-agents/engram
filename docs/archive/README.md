# archive/

Historical one-time migration artifacts preserved for reference. These files ran once during early development and do not ship into new installs.

- `migration/` — DB/config schema migration scripts
- `migrate-backup.sh`, `migrate-to-plugin.sh` — scatter→plugin one-time migration tooling
- `migrate_db_trust_tier.py` — DB migration for Layer-1 trust-tier columns
- `v1-trust-tier.md` — upgrade guide for trust-tier v1
- `test_migrate_*.py` — tests for the above scripts
